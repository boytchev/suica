# Suica

Mobile 3D made simple. As simple, as this [live demo](https://boytchev.github.io/suica/examples/minimal-example.html):


```html
<!DOCTYPE html>

<script src="suica.js"></script>

<suica>
    <demo>
    <cube>
</suica>
```

Go to [**Suica.js**](https://github.com/boytchev/suica) for more details.
