# clear

The command `clear` fills the drawing canvas with the given `color`, if it is provided; otherwise it clears the canvas to transparent.

```html
HTML:
<clear>
<clear color="crimson">
<clear background="crimson">
```
```js
JS:
clear( 'crimson' );
```
